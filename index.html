Learning Unit 1: HTML Grundlagen

Kurz, stichwortartige Zusammenfassung:

  • HTML = HyperText Markup Language
  • Grundstruktur einer HTML-Seite
  • Wichtige Tags: <html>, <head>, <body>

Code-Beispiel:

<!DOCTYPE html>
<html lang="de">
<head>
<meta charset="UTF-8">
<title>Beispielseite</title>
</head>
<body>
<h1>Hallo Welt</h1>
</body>
</html>

Learning Unit 2: CSS Grundlagen

Kurz, stichwortartige Zusammenfassung:

  • CSS = Cascading Style Sheets
  • Styling von HTML-Elementen
  • Wichtige Konzepte: Selektoren, Eigenschaften, Werte

Code-Beispiel:

<style>
body { background-color: lightblue; }
h1 { color: navy; text-align: center; }
</style>

Learning Unit 3: JavaScript Grundlagen

Kurz, stichwortartige Zusammenfassung:

  • JavaScript = Programmiersprache für das Web
  • Interaktive Funktionen und dynamische Inhalte
  • Grundsätze: Variablen, Funktionen, Events

Code-Beispiel:

<script>
function begruessen() {
alert('Hallo, Welt!');
}
</script>
<button onclick="begruessen()">Klick mich</button>

Learning Unit 4: HTML-Formulare

Kurz, stichwortartige Zusammenfassung:

  • Formulareingaben vom Benutzer sammeln
  • Wichtige Tags: <form>, <input>, <button>
  • Formularmethoden: GET und POST

Code-Beispiel:

<form action="/submit" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<button type="submit">Absenden</button>
</form>

Learning Unit 5: CSS-Layout

Kurz, stichwortartige Zusammenfassung:

  • Layouts mit CSS erstellen
  • Wichtige Techniken: Flexbox, Grid
  • Responsive Design

Code-Beispiel:

<style>
.container { display: flex; }
.item { flex: 1; }
</style>
<div class="container">
<div class="item">Item 1</div>
<div class="item">Item 2</div>
</div>

Learning Unit 6: Erweiterte JavaScript-Konzepte

Kurz, stichwortartige Zusammenfassung:

  • Asynchrone Programmierung
  • Promises, Async/Await
  • APIs und Fetch

Code-Beispiel:

<script>
async function fetchData() {
let response = await fetch('https://api.example.com/data');
let data = await response.json();
console.log(data);
}
fetchData();
</script>

Learning Unit 7: Web Storage

Kurz, stichwortartige Zusammenfassung:

  • Speichern von Daten im Browser
  • LocalStorage, SessionStorage
  • Anwendungsbeispiele

Code-Beispiel:

<script>
localStorage.setItem('username', 'JohnDoe');
let user = localStorage.getItem('username');
console.log(user);
</script>

Learning Unit 8: Einführung in Frameworks

Kurz, stichwortartige Zusammenfassung:

  • Verwendung von JavaScript-Frameworks
  • Beliebte Frameworks: React, Angular, Vue
  • Grundlegende Konzepte

Code-Beispiel (React):

<script type="text/babel">
function App() {
return <h1>Hallo, Welt!</h1>;
}
ReactDOM.render(<App />, document.getElementById('root'));
</script>
